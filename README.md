# homebrew-tap

Homebrew formulae for [@kellymears](https://github.com/kellymears) CLI tools.

## Install

```sh
brew tap kellymears/tap
brew install tarot
```

## Formulae

| Formula | Description                                                              |
| ------- | ------------------------------------------------------------------------ |
| `tarot` | [Arcana](https://github.com/kellymears/arcana-cli) — a tarot reading CLI |
